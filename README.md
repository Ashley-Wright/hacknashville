nashmentor
==========

A web application that connects Nashville's Women Technologists with young women in high school and college.

Technology Stack
==========

Front End Languages
---------------------
HTML5
- Sass 3.3.6 (not SCSS and not CSS)
- JavaScript 1.8.5
- jQuery

Database
---------------------
- PostgreSQL 0.17.1 (gem)

Source Control
---------------------
- Git
- GitHub

Deployment
---------------------
- Amazon Web Services
- Heroku

Back End Language Choices
---------------------
- Ruby 2.1.1
- Rails 4.1.0
